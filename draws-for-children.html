Doodle

Drawing tool for children and adults

It is proved that drawing has a positive effect on children by developing fine motor skills, imagination and memory. For adults, this hobby helps to relieve stress and relax emotionally. The functionality drawing application is simple and intuitive, which allows you to start drawing immediately after installation. The toolbar already has a set of colors for quick access, if it turns out to be small, you can always use the color palette with flexible settings. Among the functions of the app are: resizing the brush, saving the picture in the gallery, eraser, cleaning the canvas, the ability to add a photo or drawing from the gallery or camera, draw on the photo, undo the last actions, return the canceled actions, share the pictures on social networks. The application is adapted to different sizes and screen resolutions, i.e. it can draw with a finger on the screen of both the tablet and the phone. The application is intended for children and adults.
